Frequently Asked Questions

Can the security layer be disabled?
No. The Data Integrity Protocol is a permanent requirement for this session to ensure all users operate under the same logic constraints.
Does this monitor my private files?
Absolutely not. The protocol only intercepts keyboard and browser-specific events (like F12 or Right Click) within the scope of this web page.
Can visitors to the site ask the site owners for the security code to prevent clicks?
Yes, if website visitors ask the owners and they release the code, it can be used on other websites because code cannot be copyrighted.
